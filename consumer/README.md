# מחשבון קרן השתלמות לצרכן

## הפעלת GA4

יש להזין את ה־Measurement ID פעם אחת בלבד בשדה `gaMeasurementId` בקובץ `consumer/config.js`. כל עוד השדה ריק, סקריפט GA4 אינו נטען ולא נשלחים אירועים. גם לאחר הזנת מזהה, Analytics נטען רק אחרי בחירה ב־“מאשר/ת” בבאנר העוגיות. להפעלת פלט פיתוח מפורש בלבד, יש לשנות זמנית את `analyticsDebug` ל־`true`.

אין להוסיף לאירועים סכומי הכנסה, הפקדה או צבירה, מספר טלפון, פרטי זיהוי או תוכן הודעות WhatsApp.

## מעבר עתידי לדומיין ממותג

בעת מעבר לדומיין כגון `check.roeir.co.il`:

1. לעדכן את `publicBaseUrl` ב־`consumer/config.js`.
2. לעדכן `canonical`, `og:url` וכתובות `og:image` בשני קובצי ה־HTML.
3. לוודא שתמונת השיתוף זמינה בכתובת מוחלטת בגודל 1200×630.
4. לעדכן את כתובת האתר ב־GA4 ולהשאיר את ה־Measurement ID במקום היחיד ב־Config.
5. לבדוק שנתיבי התמונות והקישור למחשבון המקצועי עובדים גם מחוץ ל־GitHub Pages.
6. להשאיר את כתובת GitHub Pages זמינה עד השלמת DNS והאימות, ולהגדיר הפניה רק לאחר בדיקה מלאה.

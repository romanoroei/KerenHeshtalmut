# מחשבון קרן השתלמות לעצמאים v4

## תכונות עיקריות
- **טאב מחשבון צבירה** — שני מצבים:
  - הוראת קבע חודשית + הפקדה חד פעמית (עם ברירות מחדל חכמות)
  - הפקדה שנתית בתקרה
- **סיכום שנתי** — מציג תמיד את ההפקדה החד פעמית + הוראת הקבע בשנת מס ראשונה + סה"כ
- **ברירות מחדל חכמות**: אם הופקדה תקרה מלאה → SO מינואר שנה הבאה; אם חלקי → SO מהחודש הבא
- **אזהרת אי-ניצול** — מוצגת מעל "פרמטרים כלליים", מציגה רק "נשאר לניצול: ₪X"
- **טאב הטבות מס** — חישוב הטבת מס הכנסה לפי הכנסה חייבת ומדרגות מס
- כרטיסיות תוצאות מתחת לכפתור "חשב עכשיו" (מוסתרות עד לחיצה)
- ממשק RTL עברי מלא עם Heebo Font

## לוגיקת מחשבון הטבות מס
1. בסיס = min(הכנסה, תקרת הכנסה קובעת)
2. הפקדה מוכרת = min(4.5% × בסיס, ניכוי מרבי)
3. מס שולי = לפי מדרגות מס מיגיעה אישית לשנה הנבחרת
4. הטבת מס = הפקדה מוכרת × מס שולי%
5. הפקדה נוספת פטורה = תקרה מוטבת − הפקדה מוכרת

## לוגיקת הלחשבון — מחשבון צבירה
- **ריבית דריבית חודשית**: `monthlyRate = (1 + netAnnual/100)^(1/12) - 1`
- **תשואה נטו**: `netAnnual = annualReturn - mgmtFee`
- **מועד נזילות**: `31.01.(taxYear + 6)`
- **הפקדות**: מתחילות מחודש ההפקדה החד פעמית

## עדכון שנתי
עם פרסום ספר הניכויים החדש, יש לעדכן ב-`index.html`:
1. הוסף ערך חדש ל-`CEILING_DATA` (ceiling, taxableIncome, maxDeductible)
2. הוסף מדרגות מס חדשות ל-`TAX_BRACKETS`
3. עדכן את שדה year selector בטאב הטבות מס
